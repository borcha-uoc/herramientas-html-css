Frameworks

Los frameworks ofrecen un conjunto de herramientas, librerías, configuraciones y buenas prácticas para facilitar el desarrollo de aplicaciones.

Meteor

Meteor es un framework web JavaScript isomorfico que permite la creació nde rápidos prototipos y produce código multiplataforma (Android, iOS, Web). Además se integra con MongoDB y uso un protocolo de datos distribuidos que mediante un patrón de publicación-subscripción propaga automáticamente los datos entre el servidor y los clientes sin necesidad de crear ningún código de sincronización de datos.
Next.js

Es un framework basado en React, Webpack y Babel en el que se integran con un conjunto de herramientas que nos permite crear aplicaciones estática, con renderizado en el lado servidor con un gran rendimiento de forma muy simple y con una configuración mínima.
Ember.js

Ember es un framework front-end JavaScript diseñador para ayudar a construri website con interacciones de usuarios complejas. Aporta funcionalidades a los desarrolladores para poder gestioanr la complejidad de las modernas aplicaciones web.
Vue.js

Vue.js is un framework JavaScript basado en el patrón Model–view–viewmodel para construir interfaces de usuario y Single-Page-Applications. Al contrario que otros framework monolíticos, Vue está diseñado desde el principio para ser adoptable de forma incremental.
