Lenguajes

Como parte del estándar web podemos utilizar javascript en lado cliente. Sin embargo esto no nos limita a la hora de programar sitios complejos ya que podemos utiliziar otros lenguajes de programación que serán compilados/transpilados y nos permitirán aprovechar distintos características, estilos, librerías y facilidades que nos aporta cada uno de ellos.

CoffeScript

CoffeeScript es un pequeño lenguaje que transpila en JavaScript. CoffeeScript es un intento de mostrar los partes buenas de JavaScript de una forma simple.
Clojurescript

ClojureScript es un robusto, práctico, y rápido lenguaje de programación con un conjunto de funcionalidades que conforman una simple, coherente y poderosa herramienta para el desarrollo web.
Elm

Elm es un lenguaje funcional que transpila en JavaScript facilitando la creación de aplicaciones web. Hace gran énfasis en la simplicidad y la calidad de las herramientas.
Typescript

TypeScript es un lenguaje de programación desarrollado por Microsoft. Es un superconjunto de JavaScript, añadiendo entre otras características, el tipado estático.
